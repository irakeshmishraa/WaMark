# Changelog

## [1.0.0] - 2026-05-29

### Added
- Initial release of WaMark platform
- Installation wizard with 5-step setup
- Admin panel with complete dashboard
- User/client dashboard with analytics
- WhatsApp Business Cloud API integration
- Webhook handler for incoming messages & status updates
- Campaign management (broadcast, scheduled, drip)
- Contact management with CSV import
- Contact groups and tagging
- Automation engine (welcome, follow-up, birthday, anniversary, cart recovery)
- Auto-reply system with keyword matching
- Message template management
- RESTful API with key-based authentication
- Subscription billing (Stripe, Razorpay, PayPal, PayU)
- License key management and validation
- White label system (branding, colors, domain, CSS)
- Role-based access (Super Admin, Reseller, Client)
- 2FA authentication (TOTP)
- Audit logging
- Automated backups with retention
- Cron system for background processing
- Dark/light theme with responsive design
- Rate limiting and security headers
- CSRF, XSS, SQL injection protection
